# **App Name**: DoctApp UG

## Core Features:

- Doctor Search: Search for doctors by specialty and location with detailed profiles and booking options.
- Appointment Booking: Schedule appointments with available doctors through a multi-step modal.
- Health Tracking: Track health metrics such as period cycles, temperature, and blood sugar with visual charts and log entries.
- Medicine Ordering: Upload prescriptions and order medication through a streamlined process with mobile money payment option.
- Personalized Health Advice: AI-powered recommendations based on tracked health data to provide personalized advice. This tool monitors tracked health data, and determines when providing suggestions or redirecting the user to their healthcare professional might be useful.
- User Profile Management: Manage personal information, settings, and preferences, including dark mode toggle and mock sign out.
- Activity Feed: Display a feed of recent activities related to the userâ€™s health and appointments.

## Style Guidelines:

- Primary color: Emerald green (#10b981) for a fresh and natural feel, reflecting Uganda's landscape.
- Background color: Soft, warm gray (#F9FAFA), a slightly desaturated tint of emerald, to provide a clean, neutral base.
- Accent color: Soft blue (#38bdf8) to complement the green and provide visual interest for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; suitable for both headlines and body text.
- Use Lucide React icons for a consistent and clean aesthetic throughout the app.
- Mobile-first design with a bottom navigation bar for easy access to the main tabs on smaller screens, and a top bar for desktop screens.
- Subtle animations and transitions using Framer Motion (optional) to enhance user experience and provide a smooth feel.
